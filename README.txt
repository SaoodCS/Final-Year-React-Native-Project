2. RUN THE EMULATOR, AND WAIT FOR IT TO START UP
3. DRAG AND DROP THE app-release.apk FILE FROM THIS ROOT FOLDER ONTO THE EMULATOR DEVICE SCREEN

OR: 
1. FOLLOW THE INSTRUCTIONS ON SETTING UP THE REACT-NATIVE DEVELOPMENT ENVIRONMENT
2. OPEN TERMINAL 
3. CD TO THE PROJECT DIRECTORY (STRATEGYM)
4. TYPE IN THE COMMAND "npx react-native start"
5. TYPE IN THE COMMAND "npx react-native run-android"
